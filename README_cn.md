# 🌉 基于RDKX5机器人的桥梁水下露筋检测与LIMELight低照度增强
## LIMELight 低照度增强 + YOLOv11 目标检测

> 本项目面向桥梁水下结构健康检测场景，提出一套融合**水下低照度图像增强算法**与**目标检测技术**的端到端智能识别系统，完整部署于 RDK X5 边缘计算平台，实现水下钢筋裸露缺陷的高精度实时检测。
>
> 针对水下环境照度不足、光照不均、水体浑浊导致的图像质量退化，以及水下露筋公开数据集匮乏的行业痛点，项目自研 LIMELight 低照度增强算法，构建自建水下露筋数据集，并基于 YOLOv11 实现高精度检测，最终通过边缘优化在 RDK X5 上达到 13 FPS 以上的稳定推理帧率。

## ✨ 核心特性

### 🔦 LIMELight 水下低照度增强算法
基于 Retinex 理论与经典 LIME 算法针对性改进，采用**照度分区自适应、双伽马校正、高光保护、频域去模糊**四大核心机制，有效解决水下图像亮度衰减、照度不均问题。算法时间复杂度从 O(n²) 降至 O(n)，空间复杂度降至 O(1)，实现约 56 倍加速，完美适配边缘设备实时运行。

### 📦 自建水下露筋数据集
搭建可控水下实验环境，采集不同光照强度、不同水体浑浊度下的露筋图像，总计 **2944 张高质量标注图像**，覆盖多种粗细、锈蚀程度的露筋样本，填补了桥梁水下结构露筋领域公开数据集的空白。

### 🎯 YOLOv11 高精度露筋检测
基于 Ultralytics YOLOv11 框架训练检测模型，经 LIMELight 算法增强后，检测精度达 **86.41%**，mAP@0.5 达 **81.57%**，相比原始图像检测精度提升 9.13%，充分满足工程检测的精度要求。

### ⚡ RDK X5 边缘实时部署
基于 Horizon Open Explorer 工具链完成模型 INT8 量化，通过**移除冗余反量化算子、阈值反激活预筛选**策略深度优化推理性能。系统单帧总处理耗时低于 75 ms，整体帧率稳定在 13 FPS 以上，支持视频流端到端实时处理。

## 💡 技术创新点
### 1. 算法创新：
提出 LIMELight 水下低照度增强算法，通过照度分区与双伽马校正解决水下照度不均问题，大幅降低计算复杂度，实现边缘端实时增强。
### 2. 数据创新：
自主构建多条件水下露筋数据集，覆盖不同光照、浑浊度与露筋形态，填补该领域公开数据空白。
### 3. 部署创新：
在 RDK X5 平台完成算法全流程部署，通过算子移除、后处理优化等策略实现 13 FPS 以上实时推理，满足水下巡检实时性需求。
